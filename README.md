## 🚀 Proje Hakkında

**NexPOS**, restoran, kafe ve benzeri yeme-içme (HORECA) işletmelerinin tüm dijital altyapısını modernize eden, nesne yönelimli (OOP) mimariyle tasarlanmış kapsamlı bir otomasyon yazılımıdır. 

Geleneksel POS sistemlerinin karmaşık arayüzleri ve iletişim gecikmeleri gibi problemlerini çözmeyi hedefleyen bu proje; minimal öğrenme eğrisi, tam zamanlı (real-time) senkronizasyon ve şeffaf yönetim paneli sunar.

---

## ✨ Temel Özellikler

### 🤵 Kullanıcı (Garson) Paneli
- **Dinamik Masa Durumları:** Dolu, boş ve rezerve masaların renk kodlarıyla görselleştirilmesi.
- **Akıllı Sipariş Girişi:** Kategorilere ayrılmış büyük butonlar ile minimum hata payı.
- **Miktar Seçimi (Quantity Control):** Stepper (+/-) butonlarıyla anında miktar ve sepet tutarı hesaplama.

### 👑 Yönetici (Admin) Paneli
- **Gerçek Zamanlı Telemetri:** Günlük ciro, aktif masa sayısı ve sipariş istatistiklerinin anlık takibi (Dashboard).
- **Dinamik Ürün Yönetimi:** Sınırsız kategori ve ürün ekleme, anında (live-sync) fiyat güncelleme.
- **Kasa ve Finans:** Esnek ödeme yöntemleri (Nakit/Kredi Kartı) ve masa kapatma (Checkout) protokolü.

### 🛡️ Güvenlik ve Kararlılık
- **Input Validation:** Hatalı veri girişlerini önleyen form doğrulamaları.
- **Modern Bildirim Sistemi:** Ekranı kilitlemeyen animasyonlu *Toast Notification* altyapısı.
- **Güvenli Bellek Yönetimi:** Çalışma zamanı (Runtime) çökmelerini engelleyen optimize mimari.

## 💻 Kurulum ve Çalıştırma

Projeyi kendi bilgisayarınızda çalıştırmak için aşağıdaki adımları izleyin:

### Gereksinimler
- Python 3.8 veya üzeri bir sürüm yüklü olmalıdır.

### Adımlar

1. Repoyu bilgisayarınıza klonlayın:
   ```bash
   git clone https://github.com/KULLANICI_ADINIZ/nexpos.git
   cd nexpos
   ```

2. Gerekli kütüphaneyi (PySide6) kurun:
   ```bash
   pip install PySide6
   ```

3. Uygulamayı başlatın:
   ```bash
   python main.py
   ```
*(Not: Veritabanı dosyası `nexpos.db` ilk çalıştırmada otomatik olarak oluşturulacaktır.)*
